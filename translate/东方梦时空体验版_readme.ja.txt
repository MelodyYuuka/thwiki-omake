●１．ゲーム起動関係
　　　Ｑ．インストール出来ません。
　　　Ａ．インストール出来ないと、これを読めません。はて？

　　　Ｑ．起動すると、"ＧＤＣクロックが５ＭＨｚです"って怒られて
　　　　　ゲームが出来ません。
　　　Ａ．えーと、このゲームはＧＤＣクロックは２．５ＭＨｚのみです。
　　　　　マシンのマニュアルを見て切り替えてください。
　　　　（Ｈｅｌｐキーを押しながら、リセットするとメニューが出てくる
　　　　　と思ったけど・・・　９８全部同じなのかな？）
　　　　　なお、Ｗｉｎｄｏｗｓ９５のＤＯＳプロンプトでのプレイは出来ま
　　　　　せんので、Ｗｉｎｄｏｗｓ９５専用マシンでプレイしたい場合、
　　　　　ＭＳ－ＤＯＳモードで起動（Ｗｉｎｄｏｗｓ９５のマニュアルを
　　　　　お読み下さい）してプレイして下さい。

　　　Ｑ．起動すると"メモリが足りねー"って怒られて、ゲームが出来ないんです。
　　　Ａ．空きメインメモリを増やして下さい。
　　　　（このゲームはメモリは少ない方なので、これが実行出来ない位だと
　　　　　ちょっと空きが少なすぎなのでは？）

　　　Ｑ．ＣａｎＢｅシリーズでプレイしたのですが、起動出来ません。
　　　Ａ．ＣａｎＢｅシリーズの内蔵音源ですと、ＦＭ音源ドライバを常駐時
　　　　　に止まってしまうようです。お手数ですが、CANBE.exeを実行し、
　　　　　ゲームをプレイするときは、GAME[ﾘﾀｰﾝ] でなく GAMECB[ﾘﾀｰﾝ] 
　　　　　と入力してください。
　　　　　それでも、まだ同じとこで止まるようでしたら、GAMECB.BATの
　　　　　　pmdppz /M8 /V0 /E2 /K 
　　　　　の所を
　　　　　　pmd /M8 /V0 /E2 /K 
　　　　　に、さらに
　　　　　　pmdppz /R
　　　　　の所を
　　　　　　pmd /R
　　　　　書き換えて、GAMECB[ﾘﾀｰﾝ]　と入力してください。


　　　Ｑ．なんかしんないけど、実行してもゲームが始まりません。
　　　Ａ．それ以外の原因は、えーと、機種がＰＣ９８（と互換機）じゃないとか、
　　　　　なんか、変な常駐物があるとか、えぇーと。
　　　　　ファイルが少ない、もしくは、壊れているなんてことも在りますね。
　　　　　　　　インストールすると次のファイルが作成されます。
　　　　　　GAME    .BAT    ・・・　起動用バッチファイル
　　　　　　PMD     .COM    ・・・　ＦＭ音源ドライバ　Ｐ．Ｍ．Ｄ．
　　　　　　PMD86   .COM　　・・・　　　　　　同上
　　　　　　PMDB2   .COM　　・・・　　　　　　同上
　　　　　　ZUN     .COM    ・・・　音源チェックやその他雑用プログラム
　　　　　　MAIN    .EXE    ・・・　プログラム本体
　　　　　　MAINL   .EXE    ・・・　　　　　　同上
　　　　　　OP      .EXE    ・・・　　　　　　同上

　　　　　　夢時空1 .DAT    ・・・　このゲームのデータ
　　　　　　夢時空2 .DAT    ・・・　このゲームのデータ
　　　　　　夢時空　.TXT    ・・・　このゲームのマニュアル
　　　　　　夢の人々.TXT    ・・・　キャラ別性能表
　　　　　　おまけ  .TXT    ・・・　おまけテキスト
　　　　　　README  .TXT    ・・・　このファイル
　　　　　　CANBE   .EXE    ・・・　ＣａｎＢｅの内蔵音源でプレイする場合用

　　　　さらに、ゲームを実行すると、次のファイルが作成されます。
　　　　　　YUME    .CFG    ・・・　ゲームのオプション等が記録されてます。
　　　　　　YUME    .NEM    ・・・　ハイスコアが記録されてます。

　　　　しなおしてください。
●２．敵の白弾関係
　　　Ｑ．なにもしてないときに降る白弾は、どこから降ってるんでしょうか？
　　　Ａ．それは、相手からの攻撃です。よーくみると、相手フィールドから
　　　　　自分のフィールド画面の画面上に向かって半透明の弾らしきものが
　　　　　飛んでいってます。それが、画面上に付いたとき白弾となって降っ
　　　　　てくるのです。

　　　Ｑ．どうやったら相手に白弾を送れるの？
　　　Ａ．敵を連爆させるか、敵の爆発で白弾を消すと送る、または送り返す
　　　　　ことが出来ます。送り返したものは多少、弾の速度があがります。
●３．連爆システム関係
　　　Ｑ．突然、コンボ数が跳ね上がるときがあるんですが・・・？
　　　Ａ．白弾一つ送り返したのも１ヒットになります。
　　　　　よって、ボスやゲージアタックなど、白弾が大量に発射させられる
　　　　　ところで連爆すると、かなり跳ね上がります。
　　　　（最高２５５ヒットなので、それを目指しましょう）


　　　Ｑ．連爆ポイントがよくわからないです。連爆が途切れても増加しますが？
　　　Ａ．これは、連爆すると現れる画面左上のボーナス点のことですが、
　　　　　得点が点滅している時に新たな連爆をすると、継続して増加します。
　　　　　つまり、テンポよく連爆させることでかなり稼ぐことが出来ます。
　　　　　このポイントが、５１２００以上でボスアタック、ボスリバーサル
　　　　　は、送られてきた回数×５１２００、また、３０００００以上で
　　　　　ボスパニックが出ます（いずれも自動的に）。

　　　Ｑ．ちゃんと連爆ポイントが点滅しているときに、敵を倒しているのに
　　　　　ポイントが継続されません。どうして？
　　　Ａ．それは、倒すだけじゃ駄目です。２ヒット以上の連爆をする必要が
　　　　　あります。
●４．その他のゲームシステム関係
　　　Ｑ．ライフの減り方はどうなっているのですか？
　　　Ａ．始まってすぐに当たってもダメージは少ないですが、時間と共に
　　　　　少しずつダメージ量が増えます。また、ダメージを受けると相手
　　　　　の次のダメージ量が減ることもあります。
　　　　　重要なのが、最後は必ずハート半分が残ります。つまり、ハート
　　　　　が半分だけじゃ無い場合、最低でも次では死なないということです。

　　　Ｑ．コンピュータをみると、たまに明らかに弾を貫通してるときが
　　　　　あるんですが・・・？
　　　Ａ．それは、溜め撃ちを撃つときほんと一瞬（０．１秒位）ですが、
　　　　　無敵時間をもってます。人間はこれを利用することなんてほぼ
　　　　　無理ですが、コンピュータは、それで弾を抜けたりするときが
　　　　　あります。なお、コンピュータもいかさまはしてませんので、
　　　　　人間もまったく同じように動けばかわせるはずですが・・・

　　　Ｑ．なんか、自キャラが赤く光って無敵になるときがあるんですが？
　　　Ａ．それは、わりと忘れやすいゲージパワーアップです。
　　　　　ボムがなくなってからゲージを全て使用して無敵になれます。
　　　　　詳しくは  夢時空.txt　参考
●５．応用編
　　　Ｑ．具体的に、点数の稼ぎ方を教えてください。
　　　Ａ．重要なのが、クリアーボーナス。その中でも、ゲージアタック
　　　　　を頻繁に使うといい感じです（反撃を受ける１の方）。
　　　　　相手に反撃させて、それをまた返していく形になります。
　　　　　ただ、ラストあたりはそれをやるとクリア自体が難しくなり
　　　　　ますけど・・・
　　　　　あとは、とにかく大きな連爆を狙って下さい。運も絡みます。
　　　　　目標は５０００万！！　（大変～）

　　　Ｑ．あなたの腕前で１００点とりたいんですが。どうするの？
　　　Ａ．これは、とにかくダメージを受けない、ボムを使わない、
　　　　　（全ての面で）ことです。得点が高いとそれなりに腕前が
　　　　　上がりますが、それより減点の方を減らした方がいいです。
　　　　　（ちなみに１億点を越えると腕前も１００点になります
　　　　　　　って、無茶言うな～）
