Ｑ．无法安装
　首先确定安装硬盘有足够的空间吗？
　而且，安装路径的属性是只读吗？
　或者、因为某些理由install.exe无法正常运行的话，就把enya文件夹下的全部内容都复制到硬盘。
　之后，由于是从CD复制的，所以带有只读属性，请解除，这样安装就结束了。


　万一，初期的盘质量不好的话，请和我联络
　这种时候，希望能到多个机器（光驱）上尝试读取。
　因为有些光驱读游戏盘就是比较困难，这种情况即使更换了游戏盘，估计也一样读不出来。

　误认为是CD盘而放到播放机里，或者是误认为是八尺镜而将其供奉起来，
　这样的情况下没办法对应上述的解决方案，请予以谅解。



Ｑ．无法起动，出现了「direct 3D初始化失败，游戏不能运行」的警告
　可能是由于显卡不支持Direct3D的缘故，请尝试以下方法。
　　·访问显卡开发商主页并下载最新的驱动程序。
　　　　对应Direct3D的显卡，更有可能是显卡驱动过旧。

　　·改变画面模式为High Color(16Bit)， 使用同目录下的custom.exe来改变。
　　·OS起动后、在运行别的程序前先运行此程序。

　试过以上步骤仍然不行的话，看来就是显卡不支持Direct3D了。



Ｑ．出现「没有安装DirextX 8.0 以上版本无法运行」提示，无法启动
Ｑ．出现「dinput.dll ……」提示，无法起动
Ｑ．出现「dsound.dll 某些除数……」提示，无法起动
　你的机器可能是安装了Direct X8.0以下版本。
　请从WindowsUpdate （Microsoft主页）、或是杂志附赠的光盘中安装DirectX 8.0以上版本。



Ｑ．出现「不能同时运行此程序」提示，无法起动。
　已经运行「东方永夜抄」的时候，注意不要再双击图标。
　或者，万一出现异常关闭，再运行游戏时发生以上情况，请重新起动OS。



Ｑ．出现「Direct3D 对象因为不明原因无法作成」提示并且不能运行。
　原因不明的失败。因为可能是由于DirectX没有正常安装导致其无法运行。
　所以请重新安装DirectX。



Ｑ．出现「无法读取？？？，数据丢失或是损坏」提示并且自动结束游戏
　程序文件可能损坏了，错误安装东方永夜抄可能会出现此错误。请重装游戏。




Ｑ．出现「文件不能写出」提示然后结束
　文件夹或是文件属性可能被设置为写入禁止，或是硬盘空间不足，安装后改掉禁止写入属性即可。




Ｑ．如何卸载游戏？
　删除游戏文件夹即可。
　游戏安装时并不会往注册表里写什么东西，也不会有别的文件存在，记得删除自己建立的快捷方式。



Ｑ．标题画面的logo、游戏中的子弹变成白色正方形
Ｑ．标题画面的logo、游戏中的子弹变成黑色正方形
Ｑ．无法看到标题画面的logo、游戏中的子弹等
　这是显卡使得多边形顶点色彩合成不顺利。
　在安装文件夹下运行custom.exe。选择不使用顶点缓冲试试



Ｑ．标题画面的logo等显示为灰色
Ｑ．游戏中画面很暗
　这个主要是显卡的雾化效果不正常(主要是驱动程序导致)引起的。
　在安装文件夹下运行Custom.exe。选择不使用雾化效果试试。



Ｑ．不能保存录像
　画面上出现「continue」并且选择了是的情况
　或者是出现了 Slow 模式不能保存Replay 的提示，如果还是不能保存，请看下面几种状况。
　·OPTION 中将Slow模式调到了ON吗？
　·没有使用变更处理速度的程序吗？
　·没有使用任何的Debug工具吗？


　任何以上条件都可能造成游戏异常，
　请将同时运行的后台程序关闭或是更换处理落率低的机器来进行游戏。
　应该会有所改善。
　



Ｑ．游戏全部都是超高速运行
　不可思议的力量使得VSNC无法保持同步了
　OS和显卡兼容性的问题，按照以下方法变更设置。
　「开始 -> 设定 -> 控制面板 -> 显示-> 设定 -> 详细 -> ... 」
　在这里设定Direct3D，选择垂直同步(或者VSync)
　请找出那个选项并修改。

　将这里的选项『自动设定（Appli设定）』设置为『通常ON』



Ｑ．游戏画面以外的部分出现奇怪的格子、角色身上有横线
　全屏抗锯齿机能为ON。
　OS和显卡兼容性的问题，按照以下方法变更设置。
　「开始 -> 设定 -> 控制面板 -> 画面 -> 设定 -> 详细 -> ... 」
　Direct3D设定中，在这里请更改全屏抗锯齿（或是SmoothVision）设定，请找出那个选项并修改。

　将这里的选项『自动设定（Appli设定）』改为『通常OFF』



Ｑ．处理落率好高啊
　运行custom.exe,设置描画间隔为1/2或1/3。
　或是将画面设置改成『16Bits』，『16bits纹理合成禁用』再试试。

　这个游戏一定程度上取决于你的显卡配置，所以无论CPU多么的快，
　内存多么的大，都是没有意义的，请注意这点。



　
Ｑ．音乐无法正常播放
　处理困难的时候，音乐将有可能无法播放。
　这时请在Custom.exe中选择『BGM由内存读入』
　读取可能会花费点时间，但是可以稍微减轻播放的错误。
　不过也有选了还是不能正常播放的情况，这样的话还是请更换处理落率低的机器玩吧。
　处理落率低的时候音乐还是不正常的话，可能是因为同时开了别的应用程序，请硬盘加速或是关闭别的程序再试试吧。



Ｑ．想要消去记录的分数，或是想保存起来。
　将安装文件夹中Score.dat文件删除即可使游戏初始化。
　分数，通关情报，EX关卡等内容都会被初始化

　想要保存的话就请保存SCORE.dat即可



Ｑ．不能斜向移动，不能放bomb
　Dos/V键盘，有些键同时按下是没有反应的
　键盘的原因看来是没法解决了。
　所以，还是推荐用手柄来玩这个游戏。




Ｑ．和设定的不同，启动后发生严重拖慢
　开始就是0.1FPS,然后等了很久才进入开始画面。
　这样的话，按下Alt＋F4退出程序，关闭窗口吧。



Ｑ．分数在某一天突然消失了。
　改变游戏版本的话可能会使得分数初期化。
　不过也有正常初期化的时候，这时就有可能是病毒的问题了。
　从这次起，想要取回备份好的分数文件的话。
　在backup文件夹中将名为 score??.??????.bak 的文件保存下来（限制5次）。
　万一分数消失的话，可以从这里取回丢失的分数。



Ｑ．文字模糊看不清楚
　由于显卡原因，文字可能显示不出来、也存在着文字下部破损的情况。
　这是硬件的原因，没法回避、如果是由于显卡驱动的话，请更新显卡驱动。



Ｑ．画面晃动，振动，摇摇晃晃。
　无法取得垂直同期的原因。
　窗口模式无法解决此问题。
　全屏的话，解决此问题的方法是在custom.exe里选择『不使用垂直周期』选项
　或者，有些地方和游戏本身并不合拍。




Ｑ．输入会延迟一拍
　程序是不会延迟1int以上的、可能是以下几种原因。
　输入接口用的是USB的话、想要游戏安稳运行需要足够的环境来支持。

　效果音发音延迟，游戏动作也变得很慢。
　这种时候，请检查声卡的缓冲机构。
　人物的动画部分感觉也会拖慢。
　这种时候一点也不习惯呢^^;
　由于内部变化的迅速，拖慢的时候可以用眼睛看出来。

　由于显卡拖慢的问题，请选择「不使用垂直同期」来改善、
　像这类的报告,不过，为了不会出现更加致命的缺陷，除没明显地改善的情况和根本没必要改善的情况之外，请绝对不要启用这项。
　也有别的不确定的情况出现。
　用这方法的话，可能原因在于效果音的延迟和动画片的延迟等另外的地方，
　大概是感觉的关系吧…………

　实际上，根据CPU负荷的变化和绘画周期的变化，多少有变化的可能性,不过，如果是正常的环境，不可能超过1int的变化的。
　因为这，是人世间的全部游戏最低限度的延迟，再有延迟的话从某种程度上来说意味着危险的感觉（笑）
　如果，产生很明显地很大地变化，请试着对环境重新测试一下。




Ｑ．一旦按下暂停键画面就会变得模糊而看不清楚
　就是看不清楚，基本上，除了要停止游戏，请不要怀着其他目的按下暂停键。
　我觉得在射击当中按下暂停键，就跟玩猜谜游戏时这么作一样是一种邪道。


Ｑ．不能跳过Ending吗？
　看过一次的Ending就可以按住CTRL键跳过了
　Staffroll也一样，二周目后就可以跳过了


Ｑ．有二周目吗？
　Bomb使用次数在2以内通关也不会有二周目的。
　想玩二周目，自己第二次去玩游戏吧。

　不过，使用的角色不续关通关的话，会有EXTRA关可以玩。
　因为设定的难度嘎，关卡，boss和曲子都是专用的，如果习惯的人就去玩吧……


Ｑ．太难了，通不了关
　可以的，大概。
　和看上去的不同，难度不是很高。（Normal为止）
　在面对难以应付的攻击时还是使用bomb吧，我认为最少可以存活3倍以上的时间，请试着努力吧。


Ｑ．EXTRA，HARD，LUNATIC简直不可能通关
　这些本不是面向大众的难度，而是为向更高目标挑战的人而作的。
　这是因为，不论练习多少此，每个人之间还是会有能力差别。
　「如果Lunatic没穿，这游戏就不算打穿」
　这种说法是没有的，劝大家还是要以自己的能力来订立目标。
　或者说，去看别人的Replay会更好些。

　还有，高难度下，敌人的子弹是高速弹和低速弹混合的状况
　高速弹，都是规律化的，而规律化也就象征着安定，因此躲避起来更加简单。
　事实上没有完全随机的子弹，即使外表看上去随机的子弹，实质上它们还是按照一定规律来发射的。
　很早能领会这条规律的人，游戏的进展也会平稳下去的。
　反过来说低速弹，因为很少有规律性，所以一般都很密集。
　最后还是要以增大练习量来说话的。

Ｑ．成品从哪里能得到呢？
　主要是在夏COMIC或是冬COMIC展上发售大部分。
　参加事项会在主页上通知。
　或是委托同人店进行贩卖。
　这边，也在全国的店铺进行委托贩卖。
　无论是哪边，一旦售卖结束请予以谅解。


　作为成品附带的FAQ是合适的吗。


Ｑ．可以使用东方的人物和设定，来进行二次创作物的制作吗


　东方系列中登场的角色，音乐，图片，全部著作权都归开发团体所有。
　但是，作品由多人制作的时候需要都支持才通过。（困难的事情m(__)m）

　团体以同人常见的手段（主页上公开、贩卖会和委托同人商店贩卖等）
　和内容（漫画、小说、游戏、CG、Cosplay等）、允许、提出、报告等自由的使用也没任何关系。
　但是，那二次创作物的版权是创作者的，一切围绕二次创作物的纠纷全部由创作者承担。

　为了免除麻烦请遵循以下条件。


　１．禁止使用Snap以外游戏中的任何数据。

　　　直接截图，图片数据和音乐数据，文本数据，程序等资源，都禁止使用。印刷，音像等制品含有这些的也禁止使用。


　２．Snapshot以外的数据，全部是自己制作的。

　　　如果要使用别人的数据的话，必须取得对方的许可，这点是必须做到的。


　３．禁止一切对该组织，以及其他著作权拥有者的侵犯其利益的行为

　　　打破以上条件的和其他著作权的人起了纠纷的话，一切责任由该组织承担。
　　　特别是，二次创作物的作者对著作齐权要予以注意，请不要让随便转载这种事情发生。

　　　该团体游戏的内容，故意散布虚假情报、具有恶意的内容招致混乱，恐慌的话，我会停止它的制造、散布。


　４．作品的出处，记述（推荐事項）

　　　团体名为「上海アリス幻樂団」「上海アリス幻楽団」哪个都没关系


　５．有条件的话，最好联络我一下（推荐事项）

　　　CG单体和音乐单体就没关系了。


　６．团体的作品中不要写错角色名（笑）。


　７．带有文字的关于结局的东西禁止转载。

　　　使用结局的内容并不是在二次创作范畴之内、文字也禁止转载。


　８．同人团体以外的企业进行的二次创作、或是二次创作物在非同人shop中贩卖、请必须先取得我的同意。
　　　在一般杂志和书籍上出现，请必须告知我。

　　　（同人介绍杂志和二次创作杂志就没关系了）


　９．不满足以上条件但是另外的已经取得我的许可的创作物，请标明许可。



　免费或是付费（发布数量，价格），有没有成人限定，独自设定，都无所谓，请制作者自由判断
　无法判断的话请和我联系。

　如果我这边判断创作物并不满足以上限制条件的话，将停止该创作物的发布，流通，请见谅。


Ｑ．制作攻略时是否可以使用Snapshot
　当然不介意，如果事后告诉我，我会很高兴的。
　使用Snapshot必须满足以下条件。
　１．不可以使用Ending等图片。
　２．可以放大缩小，但不可以改变画面的纵横比率。
　３．请不要对图盘做以下的加工
　　　改变画面颜色，添加上本来没有的敌弹，变更Spellcard的文字等都是不行的
　　　＊但是，如果写明是作为素材的话，在上面标明是素材也不要紧的。
　４．切割画面的一部分，加上水印或是说明文字什么的都是没问题的。
　
　如果不好判断的话，可以联络我，我会给你答复的。



Ｑ．可以发布录像吗
　和Snapshot同样，是没有问题的。
　但是禁止下列事项。
　１．游戏程序之外，发布修改过的录像。
　２．使用Replay来赚钱。（禁止营利）

　现在还没有变更Replay名字的工具。
　这是为了防止随便乱改。
　因此，一旦打了错误的名字，无论怎样都没法改了，请注意。


Ｑ．对正式剧透感到很为难
　ＳＴＧ剧透、正常么？
　这样考虑过、不过仔细想一下、我想的还是过于简单了。
　以后一旦出现，想回避剧透是很困难的。
　所以请调节自己。看没看到都不要在乎就是了。


Ｑ．完全不能理解她们在说什么
　基本没什么内在意思。
　和通常的读物不同、没有第三者视点的介绍，所以请自己补完。
　这边挖苦，那边立即反唇相讥。
　没什么特别的意思。
　她们的对话只是在娱乐而已。
